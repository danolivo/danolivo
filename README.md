## Recent Projects
* __Enforce sorted scan__. The feature suggests that when an SQL query includes an ORDER BY clause for a table and a LIMIT clause, it is beneficial to have a sorted scan path among the possible scan operators during query optimisation. For instance, this could involve pushing the sort operation to the outer side of an OUTER JOIN.
